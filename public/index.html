🔒 Rate Limiting (Week 4)

Global: 100/15min Login: 5/15min

Login endpoint locks after 5 failed attempts. Logs a BRUTE_FORCE_DETECTED warning in security.log.

🌐 CORS Policy (Week 4)

Whitelist Only

Only allows requests from approved origins. Unauthorized origins get a 403 CORS Violation response.

🔑 Authentication (Week 4)

API Key JWT Bearer Dual Auth

Three levels: Public → JWT only → API Key only → API Key + JWT together.

🧱 Security Headers (Week 4)

CSP HSTS X-Frame-Options

Helmet enforces Content-Security-Policy, HSTS (1 year), X-Frame-Options: DENY, and more on every response.
